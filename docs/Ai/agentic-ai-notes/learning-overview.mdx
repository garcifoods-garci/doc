# Agentic AI Learning Overview

This section is for keeping **simple, easy-to-review notes** about what I learn related to **agentic AI**.

The goal is:

- keep each topic in a separate page when needed
- make it easy to come back later
- quickly understand what I learned
- know the next step without starting again from zero

## How this section is organized

- **Overview page**: explains how to use these notes
- **YouTube video pages**: one page per important video
- **Topic pages**: separate pages for concepts like tools, memory, planning, or evaluation when needed
- **Action items**: what to read, test, or build next

## Simple review process

When I come back later, I should follow these steps:

1. Open this overview page.
2. Open the latest video or topic note.
3. Read the **Quick Summary** first.
4. Read the **Simple Steps / Workflow** section.
5. Check the **Key Takeaways**.
6. Review the **Next Steps** section.

## Note format to follow

Each learning page should try to include:

- **Source**
- **Why it matters**
- **Quick summary**
- **Simple steps**
- **Key concepts**
- **What I understood**
- **What I should learn next**
